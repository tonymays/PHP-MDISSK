Roles

This system is designed as a Roles based system. Permission sets, whether feature and/or page permissions, can be granted to Roles once they are created. This action is performed on the Permissions Page (Administration->Permissions). Individual users that would log into this system will be granted the permissions assigned to their role.

Do the following to add a role:

1 - Click the Add Button on the Roles page
2 - On the Add Role page, enter the name of the new role within the Role field
3 - Enter a brief description of the new role in the Description field
4 - And finally, click the Save button
5 - Please note that you can repeat steps 2-4 if you have multiple roles to enter.

Do the following to edit a role:

1 - Click the Role you wish to edit on the Roles page
2 - On the Edit Role page, enter the new name of the role within the Role field
3 - Enter a brief description of the role in the Description field
4 - And finally, click the Save button

Do the following to delete a role:

1 - While on the Roles page, select Delete from the Actions column drop down for the targeted role
2 - Repeat Step 1 for other roles you wish to delete
3 - And finally, click the Save button

Please note:

1 - A System role cannot be deleted
2 - There is only one system role - Super Users.
3 - Role is a required field on both the Add and Edit Role pages
4 - Description is a required field on both the Add and Edit Role pages
5 - Role is a unique field and cannot be duplicated
